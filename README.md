# Калькулятор на чистом JavaScript

Простой калькулятор, написанный на ванильном JavaScript. Проект собран с использованием Webpack и оснащен современными инструментами для обеспечения качества кода, такими как ESLint, Prettier и Husky.

### **Функционал**

*   Все базовые арифметические операции: сложение, вычитание, умножение, деление.
*   Дополнительные операции: вычисление процента, смена знака числа.
*   Обработка ввода десятичных чисел.
*   Защита от деления на ноль.
*   **Менеджер тем:** реализована возможность переключения между светлой и темной темой с сохранением выбора пользователя.

### **Запуск и работа с проектом**

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/pavelBeloded/calculator_innowise.git
    ```

2.  **Перейдите в папку проекта:**
    ```bash
    cd calculator_innowise
    ```

3.  **Установите зависимости:**
    ```bash
    npm install
    ```

4.  **Запуск в режиме разработки:**
    ```bash
    npm run start
    ```

    *4. **Сборка development-версии**
    ```bash
    npm run build:dev
    ```

6.  **Сборка production-версии:**
    (Готовые к деплою файлы появятся в папке `dist`)
    ```bash
    npm run build:prod
    ```

7.  **Проверка и исправление кода:**
    ```bash
    npm run lint:fix
    ```

### **Структура проекта**

```
/calculator-project
├── /.husky/          # Конфигурация pre-commit хуков (Husky)
├── /dist/            # Папка с готовой сборкой проекта (игнорируется Git)
├── /src/             # Исходный код проекта
│   ├── /js/          # JavaScript модули
│   │   ├── calculator.js # Класс с основной логикой калькулятора
│   │   └── ui.js     # Модуль для работы с DOM и событиями
│   ├── /styles/      # CSS стили
│   │   └── style.css # Основной файл стилей с темами
│   ├── index.html    # HTML-шаблон для Webpack
│   └── index.js      # Главный файл, точка входа в приложение
├── eslint.config.mjs # Конфигурация ESLint
├── .gitignore        # Файлы и папки, игнорируемые Git
├── .prettierrc.js    # Конфигурация Prettier
├── package.json      # Список зависимостей и скриптов проекта
└── webpack.config.js # Конфигурация сборщика Webpack
```
